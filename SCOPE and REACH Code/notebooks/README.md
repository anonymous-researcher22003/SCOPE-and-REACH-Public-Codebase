# ETHOS-ARES Notebook Files

## File Locations

In order to add these files to the ETHOS-ARES code base, the relative path of each file will be listed with its name.

---

## Basic File Descriptions

### `notebooks/icu_admission.ipynb`
Notebook used to generate ICU admission task results and figures from inferred patient timelines.

### `notebooks/mortality.ipynb`
Notebook used to generate hospital mortality results and figures from inferred patient timelines.

### `src/ethos/metrics.py`
Modified metrics python file that now allows for the computation of SCOPE and REACH estimators while also allowing for sampling of specific numbers of timelines for estimators.
